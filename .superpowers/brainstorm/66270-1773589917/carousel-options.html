Which style feels right for the Movimento hero section?

A — Responsive 16:9

Image uses a fixed aspect ratio (16:9) instead of fixed height. Scales fluidly with screen width. No awkward cropping — content area below it.

B — Full Bleed + Text Overlay

Image fills the entire mobile screen. Text and CTA sit on top with a dark gradient. Immersive, editorial feel. Image and content share the same space.

C — Text Above, Rounded Image Card

Text and CTA come first, then a rounded-corner image card below. Clean hierarchy, no cropping issues. Modern app-like feel.
